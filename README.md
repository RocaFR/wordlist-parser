A simple parser for your wordlists :)

## Installation

```bash
git clone git@github.com:RocaFR/wordlist-parser.git
```

```bash
pip install -r requirements.txt
```
## Requirements
- `Python > 3.6`

## Usage
```bash
python wordlist-parser.py
```
